List of Interstellar Modules  [Interstellar]

Development of the interstellar project has been paused. Please check out the JS Stellar SDK docs for how integrate with Stellar using JavaScript.

Modules

  • interstellar-core: core JS wrapper
  • interstellar-network: network services
  • interstellar-network-widgets: widgets for network services
  • interstellar-wallet: services for stellar-wallet
  • interstellar-sessions: browser sessions for interstellar-client

Apps

  • interstellar-client: reference client

Complementary repositories or modules

  • solar: css framework for web applications on the Stellar network
  • solar-stellarorg: Stellar.org’s theme for Solar

Helpers

  • interstellar: command line interface
  • generator-interstellar
  • interstellar-workspace: quick start to developing interstellar-client
