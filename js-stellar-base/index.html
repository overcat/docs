Overview

The js-stellar-base library is the lowest-level stellar helper library. It can be used in a browser or in node.js. It is mainly used to read, create and sign transactions. This library is used and exposed by js-stellar-sdk but you can use it by itself if you don’t need to communicate with Horizon.

  • Building Transactions

js-stellar-base exposes the following classes:

  • Account
  • Asset
  • Keypair
  • Memo
  • Operation
  • Transaction
  • TransactionBuilder
